## 📖 详细使用教程

### 🚀 第一步：启动程序

双击 `Tron.exe` 打开主界面。

> ⚠️ **重要提示**：
> - 如果程序无法启动，请确保已安装 [Visual C++ Redistributable](https://www.microsoft.com/zh-cn/download/details.aspx?id=48145)
> - 程序必须放在英文路径下运行（路径请勿包含中文、特殊字符及空格。）

---

### ⚙️ 第二步：参数设置（重要！）

<p align="center">
  <img width="100%" src="/software1.png?raw=true"/>
</p>

#### 📝 参数详解：

| 配置项 | 说明 | 推荐值 |
|:------:|:-----|:------:|
| **目标地址** | 你想要模仿的钱包地址，或选择规则文件 | 必填 |
| **前缀位数** | 匹配地址开头的字符数量 (0-10) | 0 或 2 |
| **后缀位数** | 匹配地址结尾的字符数量 (0-10) | 6-8 |
| **生成数量** | 生成多少个地址后自动停止 (0=不限) | 1 |
| **GPU 设备** | 选择使用的显卡 | Auto |
| **输出文件** | 结果保存的文件路径 | 可选 |

---

### 🎯 第三步：两种使用方式

#### 方式一：单地址匹配（推荐新手）

<p align="center">
  <img width="100%" src="/software3.png?raw=true"/>
</p>

**操作步骤：**

1. 在「目标地址」输入框中，输入一个 TRON 地址（以 T 开头的 34 位地址）
   ```
   例如：TG2CMGxnTPgQ6V58kiKd7wbyN8ewtAmY76
   ```

2. 设置匹配规则：
   - **前缀位数**：输入 `2`（表示新生成的地址前2位要和目标地址一样）
   - **后缀位数**：输入 `6`（表示新生成的地址后6位要和目标地址一样）

3. 点击 **「开始生成」** 按钮

4. 等待生成完成，结果会显示在「生成结果」区域

---

#### 方式二：批量规则文件（进阶用户）

**操作步骤：**

1. 创建一个文本文件（如 `rule.txt`），每行写一个目标地址或规则：
   ```
   TTTTTTTTTTZZZZZZZZZZ
   TTTTTTTTTT8888888888
   TG2CMGxnTPgQ6V58kiKd7wbyN8ewtAmY76
   ```

2. 点击「选择文件」按钮，选择你创建的 `rule.txt`

3. 设置匹配规则：
   - **前缀位数**：输入 `0`
   - **后缀位数**：输入 `6` 或 `8`
**示例：**
```
目标地址：TTTTTTTTTT8888888888

前缀0位 + 后缀6位，生成结果：
TGxxxxxxxxxxxxxxxxxxxxxxxxxxxxx888888
↑↑                              ↑↑↑↑↑↑
前0位匹配                        后6位匹配
```
4. 点击 **「开始生成」** 按钮

---

## 💡 新手必读

### ⭐ 首次使用建议

**强烈建议**第一次使用时按以下设置：

| 参数 | 建议值 | 原因 |
|:----:|:------:|:-----|
| 前缀位数 | `0` | 降低难度，快速出结果 |
| 后缀位数 | `6` | 6位后缀几分钟内可出结果 |
| 生成数量 | `1` | 生成1个就停止 |

**为什么？**
- 后缀位数越高，计算时间越长
- 6位后缀 ≈ 几分钟
- 7位后缀 ≈ 几十分钟
- 8位后缀 ≈ 几小时
- 10位后缀 ≈ 几天

**建议流程：**
1. 先用后缀 6 位测试，确认程序能正常输出
2. 确认正常后，再根据需要提高位数

---

### 🎨 高仿地址建议

想要生成看起来像某个地址的"高仿地址"：

| 目标 | 前缀位数 | 后缀位数 | 难度 |
|:----:|:--------:|:--------:|:----:|
| 简单高仿 | 2 | 4 | ⭐ 容易 |
| 中等高仿 | 2 | 6 | ⭐⭐ 中等 |
| 高级高仿 | 2 | 8 | ⭐⭐⭐ 困难 |

**示例：**
```
目标地址：TG2CMGxnTPgQ6V58kiKd7wbyN8ewtAmY76

前缀2位 + 后缀6位，生成结果：
TGxxxxxxxxxxxxxxxxxxxxxxxxxxxxxtAmY76
↑↑                              ↑↑↑↑↑↑
前2位匹配                        后6位匹配
```

## ❓ 常见问题

<details>
<summary>👉 点击展开常见问题</summary>

### Q1：程序启动后闪退或打不开？
- 检查是否安装了 [Visual C++ Redistributable](https://www.microsoft.com/zh-cn/download/details.aspx?id=48145)
- 检查是否安装了 [NVIDIA显卡驱动](https://www.nvidia.cn/drivers/)
- 确保程序放在英文路径下（路径请勿包含中文、特殊字符及空格。）


### Q2：生成速度很慢或卡住？
- 检查显卡驱动是否最新
- 尝试在 GPU 设备中手动选择独立显卡
- 降低后缀位数，减少计算量

### Q3：集显和独显冲突怎么办？
- 在 GPU 设备下拉框中手动选择独立显卡
- 不要选择"Auto"，直接选择具体的显卡型号


### Q4：如何导入生成的钱包？
1. 打开 TronLink 或其他波场钱包
2. 选择"导入钱包"
3. 选择"私钥导入"
4. 粘贴生成的私钥
5. 完成导入

</details>
